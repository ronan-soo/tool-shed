# **App Name**: ToolShed

## Core Features:

- Landing Page: Simple welcoming page with links to the available tools.
- String Processing Pipelines: Visual editor to define a processing pipeline. Takes a string as input, transforms it through a series of configurable blocks, and outputs the final result.
- Escape/Unescape: String transformation block with options for HTML Entities or URI Component.
- Parse JSON: String transformation block that converts a string to a JSON object.
- Parse XML: String transformation block that beautifies/formats XML strings with custom indentation logic.
- Select Field: String transformation block that extracts data using a dot-notation path (e.g., user.data[0].id).
- Stringify: String transformation block that converts JSON objects back to strings.
- Minify: String transformation block that removes whitespace from JSON or text.
- Case Transform: String transformation block that applies uppercase/lowercase transformations.
- AES Encrypt/Decrypt: Page that encrypts or decrypts a text given the AES algorithm and a secret phrase.

## Style Guidelines:

- Primary color: Dark Slate Blue (#483D8B), offering a sense of sophistication and trustworthiness.
- Background color: Light Gray (#D3D3D3), providing a neutral backdrop that emphasizes the tool's interface.
- Accent color: Steel Blue (#4682B4), used for interactive elements and highlights, complementing the primary color with a touch of vibrancy.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, paired with 'Inter' (sans-serif) for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, outline-style icons representing each tool and block, ensuring clarity and ease of understanding.
- Clean, grid-based layout to ensure a structured and user-friendly experience.
- Subtle animations for feedback and transitions, enhancing the interactive feel without overwhelming the user.
